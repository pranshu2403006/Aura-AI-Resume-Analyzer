# Aura AI Resume

Aura AI Resume is an AI-powered resume builder and analyzer designed to help users create professional resumes and improve their chances of getting shortlisted by recruiters.

## Features

- AI-powered resume generation
- Resume analysis and suggestions
- Professional resume templates
- User-friendly interface
- Fast and responsive design
- PDF resume download

## Technologies Used

- HTML
- CSS
- JavaScript
- React (if used)
- Node.js (if used)
- AI Integration
